# Sistema de Controle de Entrada e Saída (Node.js + PostgreSQL)

Interface web em **verde e branco** no padrão do Exército

## Requisitos

- Docker ou Podman instalado
- Porta **3000** liberada na rede interna

## Como executar localmente

Basta executar via Docker:

```bash
docker compose up
```

Ou:

```bash
podman-compose up
```
